# Release Copy

These snippets match the current ArgyllUX release model: open source, AGPL-licensed, scanning for an existing ArgyllCMS install first, then guiding users to the official upstream install flow.

## Short Description

ArgyllUX is an open-source desktop app for printer profiling and print troubleshooting, compatible with a locally installed copy of ArgyllCMS.

## Repository / Website Copy

ArgyllUX is an open-source application that works with a locally installed copy of ArgyllCMS.

On first setup, ArgyllUX scans for an existing ArgyllCMS installation. If ArgyllCMS is not installed, ArgyllUX guides you to the official upstream download and setup flow.

ArgyllUX is not affiliated with or endorsed by ArgyllCMS or its author. `ArgyllCMS` is referenced only to identify compatibility with the upstream project.

## Guided Setup Copy

ArgyllUX uses ArgyllCMS for measurement and profiling workflows.

ArgyllUX will:

- scan for an existing ArgyllCMS installation
- let you choose an existing install, or
- open the official ArgyllCMS download page for guided setup

ArgyllUX does not bundle ArgyllCMS by default.
